# Installation

Theses docs require node js and some modules

```bash
npm install
```

# Document Changes

Edit the respective files in src/
- docs.java (most docs here)
- apidocs.json
- header.md
- footer.md

For syntax see:
http://apidocjs.com/


# Building docs

After changes, build the docs

```bash
./build
```
View the changes in dist/index.html

# Publication

Publish the entire dist/ folder
