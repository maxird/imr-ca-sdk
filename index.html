Welcome

Welcome to the new California IMR Document Exchange Portal Sandbox. The goal of the sandbox is to offer users an opportunity to try out the various features of the site in a test environment. Because the sandbox is hosted publicly on Amazon’s web cloud service, it is essential that real medical records are never uploaded to the site.

Accessing the Portal

In order to log into the portal, users must first be assigned a user name and password. Each Claims Administrator and Utilization Review Organization participating in the pilot must designate an in-house administrator who will manage the group’s user access. The portal offers 2 different mechanisms to provision new users. The first method permits the administrator to manually register users online. An alternative approach allows the administrator to fill out a user enrollment spreadsheet and submitting it to MAXIMUS for processing. As each new user is registered, an email notification is issued by the portal to the user with further log-in instructions.

In addition to the available screens, the document exchange portal offers web service APIs to enable system to system exchanges. An SDK is provided that contains technical documentation and sample web clients.
