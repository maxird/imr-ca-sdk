CA IMR Document Exchange Portal Sandbox

Welcome

Welcome to the new California IMR Document Exchange Portal Sandbox. The goal of the sandbox is to offer users an opportunity to try out the various features of the site in a test environment. Because the sandbox is hosted publicly on Amazon’s web cloud service, it is essential that real medical records are never uploaded to the site.

Accessing the Portal

In order to log into the portal, users must first be assigned a user name and password. Each Claims Administrator and Utilization Review Organization participating in the pilot must designate an in-house administrator who will manage the group’s user access. The portal offers two different mechanisms to provision users. The first method permits the administrator to manually register users online. An alternative approach allows the administrator to fill out a user enrollment spreadsheet and submit it to MAXIMUS for processing. The administrator is responsible for notifying users of the site link and initial credentials.

Comments and Questions

The design of the document exchange portal is a "work in progress". Our goal is to add new features to make it as useful as possible. If you have any questions or suggested improvements, please use this GitHub issues page or coordinate your feedback through your pilot administrative contact.

Technical Documentation

In addition to the available web portal, the document exchange portal offers JSON-based web service APIs to enable system to system exchanges. An SDK is provided that contains technical documentation and sample client libraries.
